1. Cálculo de Queda de Tensão em Linha CC Elétrica

Enunciado: Um engenheiro precisa calcular a queda de tensão em um cabo de cobre que alimenta uma carga industrial. O algoritmo deve receber a corrente (I), a resistência por metro do cabo (p) e a distância (L), retornando se a queda de tensão está dentro do limite de 5% para uma rede de 220V.

Pseudocódigo

Algoritmo QuedaTensao
Início
  Ler corrente, resistencia_metro, comprimento, tensao_nominal
  queda_tensao <- corrente * resistencia_metro * comprimento
  percentual <- (queda_tensao / tensao_nominal) * 100
  Se percentual <= 5 então
    Exibir "Queda Aceitável:", queda_tensao, "V"
  Senão
    Exibir "Queda Crítica! Redimensione o cabo."
  FimSe
Fim

Código Python

def calcular_queda_tensao(I, r_m, L, V_nom=220):
    queda = I * r_m * L
    perc = (queda / V_nom) * 100
    status = "Aceitável" if perc <= 5 else "Crítica"
    return queda, perc, status

# Teste com Exemplo
v, p, s = calcular_queda_tensao(15, 0.005, 50)
print(f"Queda: {v}V ({p:.2f}%) Status: {s}")

2. Dimensionamento de Banco de Capacitores Elétrica

Enunciado: Calcule a potência reativa necessária (Qc) para corrigir o fator de potência (FP) de uma planta industrial de um valor inicial para um valor desejado.

Pseudocódigo

Algoritmo CorrecaoFP
Início
  Ler potencia_ativa (P), fp_atual, fp_desejado
  angulo_atual <- arc_cosseno(fp_atual)
  angulo_novo <- arc_cosseno(fp_desejado)
  qc <- P * (tangente(angulo_atual) - tangente(angulo_novo))
  Exibir "Potência do Banco (kVAr):", qc
Fim

Código Python

import math

def dimensionar_capacitor(P_kw, fp_in, fp_fim):
    phi1 = math.acos(fp_in)
    phi2 = math.acos(fp_fim)
    Qc = P_kw * (math.tan(phi1) - math.tan(phi2))
    return Qc

# Teste com Exemplo
val = dimensionar_capacitor(100, 0.8, 0.95)
print(f"Capacitor: {val:.2f} kVAr")
