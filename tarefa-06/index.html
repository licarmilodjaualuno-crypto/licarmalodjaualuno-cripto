Análise Estatística de Sensores Industriais – Tarefa 06

Objetivo: Coletar uma lista de leituras de temperatura de múltiplos sensores em uma esteira de fundição, identificar os valores críticos (máximos e mínimos), remover ruídos na leitura e gerar um relatório automatizado do status térmico do sistema.

Algoritmo em Python (Processamento de Listas)

# Inicialização da lista de controle térmico
temperaturas = [820, 835, 848, 791, 812, 856]

# Processamento estatístico dos dados das esteiras
temp_maxima = max(temperaturas)
temp_minima = min(temperaturas)
media_termica = sum(temperaturas) / len(temperaturas)

print("Lista de temperaturas monitoradas:", temperaturas)
print(f"Temperatura máxima registrada: {temp_maxima} °C")
print(f"Temperatura mínima registrada: {temp_minima} °C")
print(f"Média operacional do sistema: {media_termica:.2f} °C")

Relatório do Painel de Controle (Resultados Simulados)

  • Vetor de Amostragem: [820°C, 835°C, 848°C, 791°C, 812°C, 856°C]
  • Pico Crítico (Máxima): 856 °C (Operação dentro do limite de segurança do refratário)
  • Ponto de Arrefecimento (Mínima): 791 °C
  • Média do Barramento: 827.00 °C
  • Status de Estabilidade: SISTEMA ESTÁVEL
