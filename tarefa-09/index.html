Estudo completo e modelação em Python da Função Afim: f(x) = 2x + 1

1. Domínio da Função

Por tratar-se de uma função polinomial do primeiro grau (função afim), não existem restrições como divisões por zero ou raízes quadradas de números negativos.

Conclusão Matemática: O domínio é o conjunto dos números reais: D = ℝ.

# Abordagem 1: Avaliação de x
x = float(input("Digite um valor para x: "))
resultado = 2 * x + 1
print(f"f({x}) = {resultado}")

2. Raiz da Função

A raiz (ou zero) indica o ponto exato onde a reta interseta o eixo x, ou seja, quando f(x) = 0.

  • 2x + 1 = 0
  • 2x = -1
  • x = -0.5

Ponto no Gráfico: (-0.5, 0)

3. Crescimento e Construção Gráfica

O coeficiente angular é a = 2. Como a > 0, conclui-se matematicamente que a função é estritamente crescente.

Abaixo está o algoritmo em Python utilizado para mapear a reta num intervalo contínuo utilizando as bibliotecas numpy e matplotlib:

import numpy as np
import matplotlib.pyplot as plt

# Geração de 100 pontos no intervalo de -10 a 10
x = np.linspace(-10, 10, 100)
y = 2 * x + 1

plt.plot(x, y, label="f(x) = 2x + 1", color="red")
plt.axhline(0, color='black', linewidth=0.8)
plt.axvline(0, color='black', linewidth=0.8)

plt.title("Gráfico da Função de 1º Grau")
plt.xlabel("Eixo X")
plt.ylabel("Eixo f(x)")
plt.grid(True, linestyle="--")
plt.legend()
plt.show()
