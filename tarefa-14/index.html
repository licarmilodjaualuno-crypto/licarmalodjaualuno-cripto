Tratamento de Anomalias em Sensores – Tarefa 14

Filtragem de dados corrompidos em tempo real de leituras de vibração mecânica.

Cenário Industrial Manutenção Preditiva

Sensores de vibração instalados em equipamentos rotativos frequentemente emitem códigos de erro (como -999.0) devido a falhas temporárias de comunicação, reinícios ou erros de calibração.

Para evitar que a análise estatística seja gravemente distorcida por esses falsos negativos, foi desenvolvido um algoritmo para detetar e remover todas as instâncias de erro do vetor antes de efetuar o cálculo das médias operacionais.

Algoritmo em Python (Data Cleaning)

# Vetor original com ruídos/falhas de leitura
vibracao = [0.12, 0.34, -999.0, 0.28, 0.91, -999.0, 1.42, 0.67]

# Loop para remoção de todos os códigos de erro
while -999.0 in vibracao:
    vibracao.remove(-999.0)

print("Lista limpa:", vibracao)
print("Quantidade de leituras válidas:", len(vibracao))

Output do Terminal

Lista limpa: [0.12, 0.34, 0.28, 0.91, 1.42, 0.67]
Quantidade de leituras válidas: 6
