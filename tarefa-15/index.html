Regras de Negócio e Engenharia

O sistema processa uma lista de medições de potência de sinal (em dBm) aplicando os seguintes critérios de validação e classificação[cite: 36]:

  • Validação: Apenas valores entre -120 dBm e 0 dBm são aceites[cite: 36].
  • Normal: Sinal >= -85 dBm[cite: 36].
  • Alerta: Sinal entre -100 e -85 dBm[cite: 36].
  • Crítico: Sinal < -100 dBm[cite: 36].

O foco arquitetónico do sistema é a aplicação do Princípio de Responsabilidade Única (SRP), dividindo as tarefas em módulos independentes: validação, cálculos estatísticos, classificação e geração de relatório[cite: 39, 40].

def is_valid_measurement(value: float, min_dbm=-120.0, max_dbm=0.0):
    return min_dbm <= value <= max_dbm

def classify_measurement(value: float) -> str:
    if value >= -85.0:
        return "Normal"
    if value >= -100.0:
        return "Alerta"
    return "Critico"
