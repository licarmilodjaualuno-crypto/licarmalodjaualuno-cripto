Catálogo de Eventos Desportivos e Automação de Galeria de Imagens em Python

Relatório Executivo

Objetivo: Reunir informações sobre eventos desportivos de Alegrete-RS (nome, modalidade, data, local) e estruturar metadados de imagens da região.

Desafios & Soluções: A principal dificuldade foi recolher informações atualizadas dispersas nas redes sociais. Para solucionar, foi utilizada uma IA para auxiliar na estruturação dos dados (JSON) e desenvolvido um script Python para gerar componentes HTML automaticamente.

Gerador Dinâmico de Galeria (Python)

Script criado para processar os metadados JSON e gerar o HTML estruturado da Galeria.

import json

CATEGORIES = ["Cards da página inicial", "Páginas temáticas", 
              "Notícias", "Eventos", "Turismo", "Cultura", 
              "Esporte", "Educação"]

def organize_gallery(data):
    organized = {cat: [] for cat in CATEGORIES}
    for item in data:
        cat = item["category"]
        if cat in organized:
            organized[cat].append(item)
    return organized

def generate_html(organized_data):
    html = "<h1>Galeria de Imagens: Alegrete-RS</h1>\n"
    for cat, items in organized_data.items():
        if items:
            html += f"<h2>{cat}</h2>\n"
            for item in items:
                html += f"<p>Fonte: {item['source']}</p>\n"
    return html
